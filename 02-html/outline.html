Files and Stuff

Suppose we save our content outline in a text editor, as outline.txt.

And then open that in a browser using File > Open.

It looks just like the text file we started with.

But what if we rename it to .html instead of .txt?

Getting started with HTML

HTML is a text format.

The browser determines whether a file it is opening is “plain text” or HTML by looking at its file suffix. (Actually, it’s more complicated than that. But that rule is ok for us now.)

That means that you can edit HTML files in a text editor.
